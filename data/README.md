# Dataset Provenance

All binary files are raw little-endian IEEE 754 float64, no header.
Size = n_values * 8 bytes.

## data/synthetic/

Generated by `tools/gen_synthetic.cpp`. Deterministic seeds — same binary on every machine.

| File | Generator | N | Notes |
|------|-----------|---|-------|
| mult_sigma0.001.bin | geometric Brownian, σ=0.001 | 1M | low-vol stock-like |
| mult_sigma0.010.bin | geometric Brownian, σ=0.01  | 1M | typical daily stock return ~1% |
| mult_sigma0.100.bin | geometric Brownian, σ=0.1   | 1M | high-vol / crypto-like |
| add_sigma0.1.bin    | additive random walk, σ=0.1 | 1M | |
| add_sigma1.0.bin    | additive random walk, σ=1.0 | 1M | |
| add_sigma10.0.bin   | additive random walk, σ=10  | 1M | |
| mixed_sigma0.01.bin | 50% mult / 50% additive     | 1M | |
| mixed_sigma0.10.bin | 50% mult / 50% additive     | 1M | |
| edge_constant.bin   | constant 42.0               | 100k | |
| edge_ramp.bin       | monotonic 0..N              | 100k | |
| edge_sinusoid.bin   | 100·sin(2πt/1000)           | 100k | |
| edge_exp_decay.bin  | 1e6·exp(-t/N)               | 100k | |
| edge_sign_flip.bin  | alternating ±U[0.5,2]       | 100k | pathological for LNS |
| edge_near_zero.bin  | U[1e-15, 1e-10]             | 100k | near-zero inputs |

License: Generated data, no restrictions.

## data/yfinance/

Fetched by `tools/fetch_yahoo.py`. Requires `pip install yfinance numpy`.

Tickers: AAPL, MSFT, GOOG, AMZN, META, NVDA, TSLA, JPM, XOM, BRK-B
Fields: Open, High, Low, Close, Volume
Period: 20 years daily

**License:** Yahoo Finance Terms of Service. Personal/research use only. Not redistributed.
Files are .gitignored. Re-fetch with `python tools/fetch_yahoo.py`.

## data/sdrbench/ (optional)

Scientific floating-point arrays from SDR-Bench (https://sdrbench.github.io).
Subsets: Miranda velocity fields, Hurricane pressure, NYX temperature.

Download scripts / links: see https://sdrbench.github.io/#datasets
Convert to .bin: `./csv_to_bin <file.csv> data/sdrbench/<name>.bin`

**License:** Individual dataset licenses apply. See SDR-Bench for details.
